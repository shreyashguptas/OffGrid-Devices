# Beacon 2 — cinematic photo set (dark / studio)

The original dark, plinth-lit studio photography for Beacon 2. Warm amber
rim-light on a charcoal background — the moody "premium-rugged" look that the
rest of the site still uses (hero, feature cards, packaging section).

These were the 2048×2048 source frames behind the home-page zoom collage before
it was switched to the bright Ember-orange set (now in `../parallax/`). They are
**not referenced anywhere right now** — kept here, renamed for clarity, so they
can be reused (social cards, blog headers, feature sections, etc.).

> ⚠️ Most of these have a **marketing headline baked into the image**, so they
> can't be dropped in just anywhere — check the "Caption" column before reusing.
